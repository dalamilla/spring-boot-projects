# Exercise Tracker

Exercise Tracker microservice implemented on Spring Boot.

## Instructions

To start this app:

- Installing Dependencies

```
gradle dependencies
```

- Running Spring Boot Application:

```
gradle bootRun
```

- Run Spring Boot Test Application:

```
gradle test
```
